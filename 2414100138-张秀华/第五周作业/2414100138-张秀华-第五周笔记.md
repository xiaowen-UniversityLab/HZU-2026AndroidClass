1。设置播放视频的多种方式：
①raw下的资源：
MediaPlayer.create(this, R.raw.test);
②本地文件路径：
mp.setDataSource("/sdcard/test.mp3");
③网络URL文件：
mp.setDataSource("http://www.xxx.com/music/test.mp3");

2.AudioManager audioManager：Android 系统的音频管理器，专门用来控制系统音量、管理音频焦点。要实现音量调节，必须通过它来操作。

3.surfaceHolder = sfv_show.getHolder();
获取 SurfaceView 的控制器 SurfaceHolder，它负责管理视频渲染的画布（Surface）。MediaPlayer 解码后的视频帧，必须交给 SurfaceHolder 才能显示在屏幕上。

surfaceHolder.addCallback(this);
给 SurfaceHolder 添加生命周期回调，监听 Surface 的创建、变化、销毁事件，重写这三个方法：
surfaceCreated：Surface 创建完成，这里可以初始化 MediaPlayer
surfaceChanged：Surface 尺寸变化（比如横竖屏切换）时触发，这里可以调整视频比例
surfaceDestroyed：Surface 销毁，这里要释放 MediaPlayer 资源，防止内存泄漏

4.setRequestedOrientation(ActivityInfo.SCREEN_ORIENTATION_...)：强制把 Activity 方向设为横屏（LANDSCAPE= 横屏，PORTRAIT= 竖屏）；

5.隐藏系统栏，实现全屏：
// Android 11及以上：用InsetsController隐藏系统栏
if (Build.VERSION.SDK_INT >= Build.VERSION_CODES.R) {
    getWindow().getInsetsController().hide(WindowInsets.Type.systemBars());
}
// Android 11以下：用WindowManager设置全屏Flag
else {
    getWindow().setFlags(WindowManager.LayoutParams.FLAG_FULLSCREEN, WindowManager.LayoutParams.FLAG_FULLSCREEN);
}

6.恢复系统状态栏，同样分版本实现：
if (Build.VERSION.SDK_INT >= Build.VERSION_CODES.R) {
    getWindow().getInsetsController().show(WindowInsets.Type.systemBars());
}
else {
    getWindow().clearFlags(WindowManager.LayoutParams.FLAG_FULLSCREEN);
}

7.之前播放视频时，向系统请求了音频焦点；退出页面时，必须调用 abandonAudioFocus() 归还焦点，这样其他 APP才能正常播放声音。

8.AudioManager audioManager：Android 系统的音频管理器，专门用来控制系统音量、管理音频焦点
audioManager = (AudioManager) getSystemService(Context.AUDIO_SERVICE);//获取系统的音频服务实例，之后才能用它来调节音量。

9.setRequestedOrientation() 切换方向后，Activity 会重启！一重启，isLandscape = true 就变回默认值 false 了！所以按钮文字永远不同步！
要在mainfest里面加上android:configChanges="orientation|screenSize|keyboardHidden"
禁止屏幕旋转时重启 Activity，这样变量 isLandscape 才不会丢失
