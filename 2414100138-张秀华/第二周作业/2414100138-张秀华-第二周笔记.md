1.适配 Android 8.0+：动态注册时需添加 Context.RECEIVER_NOT_EXPORTED 标志，防止其他应用向该接收器发送广播，提升安全性。如下：
if (Build.VERSION.SDK_INT >= Build.VERSION_CODES.S) {
            registerReceiver(batteryReceiver, intentFilter, Context.RECEIVER_NOT_EXPORTED);
        } else {
            registerReceiver(batteryReceiver, intentFilter);
        }

2.动态注册记得在onDestroy()中反注册
3.照片获取是耗时操作，要开辟一个子线程，防止堵塞主线程
4.动态权限申请：
String permission;
if (Build.VERSION.SDK_INT >= Build.VERSION_CODES.TIRAMISU) {
    permission = android.Manifest.permission.READ_MEDIA_IMAGES;
} else {
    permission = android.Manifest.permission.READ_EXTERNAL_STORAGE;
}
检查权限：ActivityCompat.checkSelfPermission()
未授权则申请：ActivityCompat.requestPermissions()
处理申请结果：重写 onRequestPermissionsResult() 回调

5.必须在 Manifest 中声明需要的权限，否则动态申请无效：
<uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE" />
<uses-permission android:name="android.permission.READ_MEDIA_IMAGES" />
